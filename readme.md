# ScrnshotMate

一個主要針對線上課程或會議全畫面截圖後的自動化圖片裁切工具。
使用者可透過定義第一張圖片的裁切區域，自動批次裁切其他相同尺寸與解析度的圖片。

ver.1.1.0 (2026/04/27)

## 系統要求
- macOS 14.0 或更高版本
- 支援 Apple Silicon 系列晶片電腦

## 開發工具
- **Language**: Python / Swift
- **Orchestration**: Anti-Gravity (AI Agent)
- **Model**: Claude 3.5 Sonnet / Gemma 4

## 安裝說明
1. 請進入Dist資料夾，將 ScrnshotMate.dmg 下載後安裝。
2. 若遇到安全性檢查，請至「系統設定」允許開啟。
