# 自動化裁切圖片工具 (Image Cropper)

## 專案概述
開發一個主要針對線上課程或會議全畫面截圖的自動化圖片裁切工具。使用者可透過定義第一張圖片的裁切區域，自動批次裁切其他相同尺寸與解析度的圖片。

## 目標平台
- 主要支援 macOS
- 預留移植至 Windows 的可能性

## 預定使用流程與功能需求

1. **啟動與首頁介面**
   - 點開程式，顯示主介面。
2. **載入檔案**
   - 支援透過按鈕選擇圖片檔案。
   - 支援直接拖曳檔案至介面中載入。
3. **圖片預覽與檢查**
   - **預覽呈現**：以類似 Mac Finder 的縮圖方式排列，下方顯示檔案名稱與圖片原尺寸。
   - **縮放控制**：右下角提供拉桿可調整預覽圖片的大小。
   - **操作按鈕**：提供「增加檔案」按鈕與「裁切」按鈕。
   - **自動檢查**：同步檢查所有載入圖片的尺寸與解析度是否相同。將與大部分圖片不同的檔案標示出來並「預設勾選」。
4. **異常處理與確認**
   - **嚴格尺寸限制**：所有載入圖片必須為同一尺寸與解析度才能進行裁切。
   - **有尺寸異常檔案**：鎖住「裁切」按鈕。使用者必須點擊取消勾選或刪除異常檔案，使列表內只剩下單一尺寸的圖片後，才能解鎖進入「裁切」。
   - **無尺寸異常檔案**：使用者仍可在此畫面確認檔案，選擇取消載入特定檔案，或直接點選「裁切」。
5. **裁切範圍設定 (單圖範例)**
   - 進入裁切介面，視窗最大化。
   - 預設以第一張圖片（或使用者特別選擇的單張圖片）為範例，圖片等比例放大至碰到視窗邊界。
6. **裁切操作與視覺回饋**
   - 使用滑鼠點按後拖動形成長方形裁切範圍。
   - 視覺回饋：範圍內維持原明度，範圍外明度降低 50%。
7. **範圍微調**
   - 方框四角可隨意拖動，連動影響相鄰兩邊，且維持 90 度直角。
   - 方框上下兩橫邊中間各有一橫槓，左右兩豎邊中間各有一豎槓，提示可拖動該邊調整位置。
8. **輸出設定**
   - 調整完畢後，點擊「確定裁切」，跳出對話框提供兩種輸出模式：
     - **a. 對原圖片檔案裁切**（裁切後會將原檔案刪除，並在原位存入新檔，完成後提示「已將原圖片刪除，並存入新檔」）。
     - **b. 另存新檔案**，包含三個設定區塊：
       - **區塊一：選擇存檔類型**。預設為原圖片格式。可選 png, jpg, webp, pdf。若選 PDF，則將所有圖片依序合併為單一 PDF 檔。
       - **區塊二：選擇存檔路徑**。
       - **區塊三：命名規則**。
         - 非 PDF：參考 Mac Finder 的批次命名功能（取代文字、加入文字、格式化連續編號且可自訂數字位數）。
         - PDF：單一檔案命名，預設為第一張原圖的名稱，使用者可以重新命名檔案名稱。
9. **執行與完成**
   - 按下確定後開始批次裁切與存檔。
   - 處理完畢後跳出「完成」提示對話框。
